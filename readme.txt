== Documentation ==

Please read documentation in the doc/ folder to know how to use this software. 

== Required components ==

To use SemLinker, you need the following components :

- You need to install the Redis Base, available for Linux and Windows : http://redis.io/download
- You need a true Java 7 compliant VM. We recommend the Oracle One : http://www.oracle.com/technetwork/java/javase/downloads/index.html
- You need a version of the NLGbAse metadata (one is in the folder metadata) : http://www.nlgbase.org
- You need a semantic annotation API key (free) : http://wikimeta.org/product_api.html 
 
== About Wikimeta ==

Wikimeta provides content analysis and meta-data annotation tools. 
Wikimeta finds the semantic richness hidden in any content including named entities, keywords, semantic nature of concepts, nature of words or sentences. 
Wikimeta uses innovative methods to analyze your content, extract semantic meta-data: information about people, places, companies, topics, languages, and more. 
Just try it on line now for your day to day needs of text analysis, or use the API to build your own applications. 

Register and obtain your free API key to use this API caller:
